# Traffic Vision AI Engine

Standalone inference server built to deploy highly dense deep-learning analytics targeting traffic extraction parameters continuously via lightweight edge hardware metrics targeting backend execution natively.

## Implementation Technologies
* **API Framework:** `FastAPI` (Python SDK)
* **Model Pipeline:** `ultralytics YOLOv8m`
* **Object Targeting System:** Deep SORT / Native `ByteTrack` Integration
* **Data Mapping:** `NeonDB` PostgreSQL architectures mapped over `psycopg2`
* **Infrastructure Pipeline:** Hugging Face Spaces SDK mapping natively via root containers

## Core APIs
- `POST /api/v1/analyze`: Resolves heavy video boundaries calculating track parameters dynamically passing outputs explicitly onto structured PostgreSQL database endpoints natively.
- `GET /docs`: Standalone ping wrapper mirroring execution availability over to React dashboard contexts dynamically bypassing Swagger payloads correctly.
